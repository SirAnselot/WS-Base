Simple hero unit, a simple jumbotron-style component for calling extra attention to featured content or information like the following:

Docs

This is basically a setup of Bootstrap v4.0.0-alpha.5 with source scss files and slightly modified grunt config. (Grundfile.js) - please refer to the Boostrap v4 alpha docs for almost all further information.

Icons

Choose one* from two included icon fonts:

  1. entypo+ by Daniel Bruce (generated by fontello.com open source project).
    ...
  2. glyphicons pro by Jan Kovarik.
    ...
*) ... or use both if a little extra size don't matter

Icon-overviews are also included locally here overview entypo and here overview glyphicons.
To use: comment/uncomment in scss/custom.scss this @import "custom/glyphiconspro"; or this @import "custom/entypo";
